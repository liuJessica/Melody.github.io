4、安装git
-1、sudo apt-get install git
-2 配置用户名和邮箱 git config –global user.name git config –global user.email
-3 生成sshkey ssh-keygen -t rsa -C “user.email”
-4 复制sshkey 添加到github配置中
5、查看文件内容 cat 文件名
6、代码提交git
-1、进入src文件，初始化git仓库 git init
-2、新建分支 git checkout -b 新分支名称
-3、创建远程仓库 git remote add 远程地址名称 仓库地址（如果配置了sshkey 可以用git@github.com:用户名/仓库名.git）
-4、git add 文件名 将待提交文件提交到待提交区
-5、git commit -m “提交说明” 提交文件
-6、git pull 远端名称 分支名称 将远端文件与本地文件合并
-7、git push 远端名称 分支名称 将当前的提交推送到远端主机的分支上
